HTML-JS-Local-Store-Example
===========================

This example creates a simple task list that keeps a list of Homework Tasks along with a a second list of classes.
Think of it as a way to keep track of homework as a to do list. 

The real goal of this example is to give an example of working with HTML5 local storage. The example makes use of 
jQuery and jStorgae. 

Local storage allows your web pages and apps to save persitent data locally.
